# OFF WHITE — Style Reference
> Neon Glitch Arcade — stark black canvas with pixelated neon blocks

**URL:** https://off-white.valeriafrancis.com
**Refero ID:** cf49b88a-fb38-4520-8fbb-ab3efa983517
**Theme:** dark
**Category Tags:** Cyber Neon, Glitch, Arcade, Editorial

## Tokens — Colors (3 vivid neons + mono base)
| Name | Value | Token | Role |
|------|-------|-------|------|
| Midnight Void | `#000000` | `--color-midnight-void` | **Page bg + primary text** + image borders |
| Ghost Canvas | `#ffffff` | `--color-ghost-canvas` | Accent bg, secondary text |
| Muted Grey | `#aba4a4` | `--color-muted-grey` | Subtle accent text |
| **Glitch Green** | `#00fb34` | `--color-glitch-green` | **PRIMARY action bg + button text + link highlights** |

| Digital Yellow | `#fff500` | `--color-digital-yellow` | **Outline accents** for tags, dividers |

## Typography (3 fonts)
### Offwhite (Brand display)
- Sub: Arial Bold. Weights: 400, 600, 700
- Sizes: 11, 18, 20, 36, 38, 42, 48, 56, 72, 96, 180, **500px** (extreme!)

### Times (Body — SERIF)
- Weight 400. Size 16px. Traditional readable serif for contrast

### Arial (Functional)
- Weight 400. Size 13px. Small functional UI

### Type Scale
| Role | Size |
|------|------|
| caption | 11 |

| body | 36 (large body!) |

| display-sm | 180 |

## Spacing & Shapes
- Base 6px (unusual), spacious
- Scale: 40, 60, 90, 120 (only 4 values — minimal)
- Section gap: 60-120px
- Card padding: 0px (no cards)
- Element gap: **40px** (very wide)

### Border Radius (binary 0px/30px)
| Element | Value |
|---------|-------|
| **Everything not interactive** | **0px** |
| **links** | **30px** |
| **buttons** | **30px** |

**Strict rule:** Only 2 radii values exist (0px and 30px).

## Components

### Filled Brand Button (Primary CTA)
**Glitch Green bg, Midnight Void text, 0px radius (sharp).** Commands attention.

### Ghost Brand Button
Transparent bg, Glitch Green text, **30px radius**.

### Rounded Link
Glitch Green text, transparent bg, 30px radius. Same feel as ghost button.

### Hero Headline Text Block
**Offwhite font up to 500px**, Midnight Void OR Glitch Green color, **with Warning Red text border**.

### Image Collage Block
1px solid Midnight Void border, **dense overlapping collage**. No specific radius.

## Imagery
**Highly treated photography** — almost exclusively tight crops of single individual. Rectangular containers, **overlapping in dense glitchy collage** against black. **Abruptly cut, bordered with 1px Midnight Void or vivid brand accent.** Raw, fragmented, aggressive density. Decorative atmosphere over explanation.

## Layout
**Full-bleed (no max-width).** Hero: dominant central headline in oversized custom font + image collage. **Dense overlapping** photo blocks alternate with blocky text. 60-120px section gaps, no subtle dividers. **Hamburger nav only** (intentional resistance to conventional discovery). High density, frequent overlap.

## Motion Notes
- **Glitch effects on text/images** (signature)
- Image collage may animate/shift on scroll
- Hover: link/button color flicker
- Marquee-style text scrolling possible
- "Arcade pixelation" feel
- Aggressive, attention-demanding motion

## Do's
- **Midnight Void (#000) default page bg** (stark dark canvas)
- **Glitch Green (#00fb34) for ALL primary interactive** (buttons, links)
- **30px radius ONLY on Ghost/Rounded Link buttons** (no other UI)
- Offwhite font for headlines/brand, Times/Arial reserved for body
- High contrast pairings (Glitch Green on void, Warning Red highlights)
- **Image collage with Midnight Void borders** to break vertical rhythm
- Spacious 60/90px paddings, 40px element gaps

## Don'ts
- No subtle shadows or gradients (hard edges + solid fills only)
- No vivid colors (green, red, yellow) for large static decorative bg (high-contrast disruption only)
- **No additional border-radius beyond 0px and 30px**
- No system fonts only (custom Offwhite critical)
- No standard grid for imagery (dense overlapping collage)
- No subtle/low-contrast text colors
- No generic spacing (40/60/90/120 only)

## Key Insight
**The "hard glitch arcade" cyber neon archetype.** The contrast: dark base + 3 unmissable neon hues (green/red/yellow) + ONE custom display font scaling to 500px (!). Strict binary radius (0px or 30px) — no in-between. Image collage with Midnight Void borders is the visual signature — overlapping, cut, fragmented. Resistance to convention: hamburger-only nav, no max-width, dense overlap. **Glitch Green is the ONE active accent — Warning Red and Digital Yellow are "outline-only" highlights.**
