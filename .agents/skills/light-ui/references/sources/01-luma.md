# Luma — Light UI Source Notes

Source: https://styles.refero.design/style/680b0fff-26d0-45c3-808b-d571ff45eb99
Original URL: https://lu.ma
Category scan: Light UI
Theme: light
North star: Festival poster behind frosted glass.

## Essence

Luma is a light UI where the chrome becomes quiet so the hero imagery and one
brand accent can feel vivid. The interface is mostly achromatic: warm near-black
text, graphite secondary labels, ash tertiary strokes, white background, and a
dark charcoal CTA. Color is not spread across the UI. It is concentrated into a
single spectrum logo mark and one flamingo phrase inside the headline.

## Color Tokens

| Name | Value | Role |
| --- | --- | --- |
| Obsidian | `#131517` | Primary text and headings |
| Graphite | `#656768` | Secondary UI text, nav labels, muted icons |
| Ash | `#a5a6a8` | Tertiary text, icon strokes, ambient time |
| Charcoal | `#333537` | Primary CTA fill |
| Pure White | `#ffffff` | Page background and button label text |
| Ink Black | `#000000` | SVG fills and maximum contrast overlays |
| Luma Spectrum | `#099ef1` + radial spectrum | Logo accent only |
| Flamingo | `#f31a7c` | One inline headline accent phrase |

## Typography

- Single system font stack: `-apple-system`, BlinkMacSystemFont, Segoe UI.
- Sizes: `13px`, `14px`, `16px`, `18px`, `20px`, `64px`.
- Weights: `400`, `500`.
- Headline: `64px`, weight `400`, line-height around `66px`.
- Letter-spacing: `-0.016em` across sizes.

## Layout

- Max width around `1200px`.
- Two-column hero: left text/CTA around 40%, right 3D imagery around 60%.
- Navigation height around `52px`.
- Footer is a compact horizontal row.
- No heavy section bands; the first viewport is simple and direct.

## Components

- Logo wordmark: black text plus tiny spectrum star, gradient only here.
- Nav bar: white background, subtle gray links, live time badge.
- Hero CTA: charcoal pill, `15px` radius, white text.
- Event cards: white cards, `15px` radius, soft date blocks.

## Reusable Lessons

- Light UI can be mostly silent while one image carries emotion.
- One accent phrase in a headline can be stronger than a whole accent palette.
- System fonts can feel premium when spacing and hierarchy are refined.
- Avoid colorful buttons when imagery is already vivid.

## Anti-Patterns

- Do not use the spectrum gradient outside the logo mark.
- Do not make the headline bold.
- Do not spread flamingo across the whole page.
- Do not add shadows to the primary chrome.
