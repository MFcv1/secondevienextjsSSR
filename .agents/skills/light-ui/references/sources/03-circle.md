# Circle — Light UI Source Notes

Source: https://styles.refero.design/style/ab8450d9-1b42-4395-aa24-9e277f021aa1
Original URL: https://www.circle.so
Category scan: Light UI
Theme: light
North star: Galactic UI with soft glow.

## Essence

Circle mixes bright white content surfaces with soft galactic gradients and
highly rounded interactive controls. It is light UI with a deep atmospheric
stage: white cards and pill inputs remain practical, while cool gradients and
translucent panels create a modern community-product feel. The system stays
approachable by using Inter everywhere and relying on soft radius rather than
hard technical edges.

## Typography

- Inter is the sole typeface.
- Sizes from `10px` to `64px`.
- Display around `48px`, line-height `1.21`.
- Weights: `400`, `500`, `600`, `700`.

## Layout

- Max width around `1376px`.
- Full-bleed gradient hero.
- Centered headline and CTA block.
- Section gaps around `93px`.
- Alternates white, dark gradient, and tinted backgrounds.
- Uses flexible columns and 3-column card grids.

## Components

- Ghost nav button: transparent, slate border, full pill radius.
- Accent filled button: pastel accent fill, black text, full pill radius.
- Product card: white, `20px` radius, soft low-opacity shadow.
- Translucent card: white at 50% opacity, `24px` radius.
- Input: white fill, slate border, full pill radius, blue focus ring.

## Reusable Lessons

- Light UI can use a dark/gradient stage if white content surfaces remain clear.
- Full pill controls create friendliness and community softness.
- Use one typeface when the surface system is already expressive.
- Translucency should feel soft, not blurry chaos.

## Anti-Patterns

- Do not use square corners.
- Do not introduce harsh saturated backgrounds.
- Do not replace Inter with a decorative face.
- Do not use heavy opaque shadows.
