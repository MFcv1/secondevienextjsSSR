## Tokens — Colors (5 grays only)
| Name | Value | Token | Role |
|------|-------|-------|------|
| Canvas White | `#ffffff` | `--color-canvas-white` | Page bg, cards |
| Ink Black | `#000000` | `--color-ink-black` | Primary text, borders, active states |
| Medium Gray | `#818181` | `--color-medium-gray` | Muted text, placeholder |
| Faded Gray | `#9d9d9d` | `--color-faded-gray` | Tertiary text, list borders |
| Light Gray | `#d6d6d6` | `--color-light-gray` | Subtle dividers |

## Typography (multiple display serifs!)
### Onsite (Body)
- Sub: system-ui. Weight 400. Sizes: 11-40px (8 values). Letter-spacing: 0.015-0.08em

### Avec Sharp (Display) — and 5 alternates
- Sub: serif. Weight 400. Sizes: **251px**!
- Letter-spacing: 0.0010em
- **Plus: Ceno, Meso, Gestura, Rework** — all serif display alternatives at 251px

### Type Scale
| Role | Size | Tracking |
|------|------|----------|
| caption | 11 | 0.88px |
| body | 14 | 0.35px |
| heading | 26 | 0.26px |
| display-sm | 40 | 0.6px |
| **display** | **251** | 2.51px |

## Spacing & Shapes
- Base 4px, comfortable
- **Section gap: 120px** (very generous editorial)
- Card padding: 0px (transparent cards)
- Element gap: 12px

### Border Radius
**0px everywhere** (sharp).

## Components

### Ghost Button - Inactive
Ink Black text, **1px bottom border** of same color, no bg, no padding.

### Ghost Button - Muted
Medium Gray text + 1px bottom border, no bg, no padding.

### Featured Card
**Transparent bg, no borders, no shadow, 0px radius.** Content manages own spacing.

### Text Input
Transparent bg, Medium Gray placeholder, **1px bottom border only**.

## Imagery
**Large-scale typographic specimen imagery** + abstract organic colorful graphic backgrounds in hero sections. **No photography.** Minimal vector outline icons.

## Layout
Full-bleed (no max-width). Hero: large abstract bg with centered prominent type specimen + info text. **Two-column** below: text + visuals/specimen. Minimalist top nav with ghosted links + large floating brand text. **120px section gaps** create spacious editorial flow.

## Motion Notes
- Subtle hover state shifts (color/border)
- Type specimens may animate weight/style on hover
- Smooth scroll between sections
- Minimal — typography is the show

## Do's
- Ink Black text on Canvas White (max contrast)
- **Strict 0px radius EVERYWHERE**
- Ghost buttons + underlined text only
- 120px section gaps for editorial breathing
- Onsite for functional, **Avec Sharp/Ceno/Meso/Gestura/Rework** for display showcases

## Don'ts
- No saturated bg or heavy fills
- No shadows or elevation
- No rounded corners
- No contrasting accent colors for CTAs (text weight/size/borders only)
- No dense clustered layouts
- No deviation from the 6 typefaces
- No generic system fonts for links

## Similar Brands
Fonts.com, Future Fonts, Pangram Pangram, Grilli Type

## Key Insight
**The "Type Foundry Showcase" archetype.** Multiple display serifs (5 different ones!) at extreme 251px, paired with Onsite sans for ALL functional text. 120px section gaps + strict 0px radius + ghost buttons with underline = editorial purity. The display fonts ARE the imagery.
