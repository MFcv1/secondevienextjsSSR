# Retool - Expressive Brand Source Notes

## Core read

Retool is the "warm obsidian workshop" archetype: a near-black canvas, warm
off-white text, compressed light-weight typography, dark slabs instead of
shadows, muted teal washes, and a warm amber-rust hero gradient. Its expression
comes from proprietary type and material restraint rather than loud color.

This source matters because expressive-brand systems for developer tools often
fail by becoming either generic dark SaaS or cyber neon. Retool shows a third
path: workshop atmosphere, warm black, and decisive shape grammar.

## Typography

- Primary: saansFont-like sans.
- Display: weight 300 at 60-72px, tight -0.020em to -0.022em tracking.
- Body/subheading: weights 300-400.
- Labels: pxGrotesk-like, 12-14px, positive tracking.
- Authority comes from huge light text, not bold weight.

## Structure

- Base unit: 4px.
- Max width: 1200px.
- Section gap: 80-120px.
- Card padding: 24px.
- Element gap: 8-16px.
- Badges: 4px radius.
- Cards: 8-12px radius.
- Pill CTAs: 9999px.
- Non-pill buttons: 0px radius.
- No in-between button radii.

## Expressive lessons

1. Warm dark is brandable
   - Use #151515, not pure black.
   - Parchment text feels crafted.

2. Type creates ownership
   - Light, compressed, large headings feel proprietary.
   - Generic Inter bold would lose the identity.

3. Slab surfaces replace shadows
   - Cards are darker/lighter layers.
   - Elevation is material, not drop shadow.

4. Chromatic colors stay inside special effects
   - Teal washes are atmospheric.
   - Shimmer colors appear only in select phrases.

5. Shape grammar is binary
   - Pill CTA or square button.
   - Avoid medium rounded mush.

## Anti-patterns

- Pure black canvas.
- Cyber neon accents.
- Saturated colors as UI chrome.
- Heavy drop shadows.
- Generic bold headings.
- Mixed random radii.
- Hero gradient inside content panels.
