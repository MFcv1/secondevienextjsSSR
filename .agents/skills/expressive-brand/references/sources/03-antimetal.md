# Antimetal - Expressive Brand Source Notes

## Core read

Antimetal is the "electric infrastructure signal" archetype: a dramatic deep
navy/electric-blue hero, a precise near-white product dashboard below, and a
single chartreuse CTA that bridges both worlds. The expression is technical,
not playful: dot-matrix globe, infrastructure atmosphere, tight custom UI type,
and blue-tinted elevation.

This source teaches how a technical B2B brand can be expressive without
becoming childish or cyberpunk. The drama is concentrated in the hero and CTA,
then the page snaps into data-dense product daylight.

## Typography

- UI/body: abcdFont-like sans, weights 400-480.
- Display: ivarTextFont-like serif for 32px+ headlines.
- UI tracking: tight from -0.016em to -0.005em.
- Display tracking: around -0.010em.
- Display serif uses OpenType alternates for craft.
- The serif/sans contrast makes infrastructure feel high-end.

## Structure

- Base unit: 4px.
- Max width: 1200px.
- Density: compact.
- Section gap: 80px.
- Card padding: 20px.
- Element gap: 8px.
- Cards: 6/16/20px radii depending on scale.
- All buttons/pills: 9999px radius.
- Hero-to-light transition should happen once.

## Expressive lessons

1. Concentrate drama
   - Dark electric hero first.
   - Light technical UI after.
   - Do not repeat dark bands everywhere.

2. One loud signal
   - Chartreuse is only for primary CTA.
   - Never use it for icons or decoration.

3. Technical illustration matters
   - Dot-matrix globe gives atmosphere without lifestyle photography.
   - Product dashboard does the proof work.

4. Shadows can be branded
   - Use blue-tinted shadows/rings.
   - Avoid generic black shadows on light product cards.

5. Typography can humanize infrastructure
   - Display serif adds craft without softening the product UI.

## Anti-patterns

- Chartreuse decoration.
- Multiple dark sections after hero.
- Generic black shadows.
- Default tracking on custom type.
- Non-pill buttons.
- Cyber neon instead of technical signal.
