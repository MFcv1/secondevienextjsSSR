# Playful — Playful Design Source Notes

Source: https://styles.refero.design/style/f93ac72e-73b2-4b2c-80eb-351ddfa56f4d
Original URL: https://playful.software
Category scan: Playful Design
Theme: light
North star: Gradient Playground.

## Essence

Playful is refined play: warm off-white canvas, vivid pink interaction, spacious
sections, compact Inter typography, large rounded dark cards, and soft gradients.
It is playful without becoming childish because the typography and layout remain
controlled. Color is concentrated into pink actions and gradient atmosphere.

## Typography

- Inter is primary for all text.
- Arial is used for robust inputs/buttons.
- Display: `79px`, line-height `1`.
- Heading: `70px`, line-height `1.15`.
- Body: `16px`, line-height `1.2`.
- Weights range from `400` to `900`.

## Layout

- Max width around `1500px`.
- Full-bleed gradient hero.
- Centered headline and input/button pair.
- Alternating centered text blocks and feature grids.
- Section gap around `113px`.

## Components

- Primary CTA: Vivacious Pink fill, Frost Canvas text, `99px` pill radius.
- Ghost button: transparent, Midnight Ink text.
- Circular secondary button: Frost Canvas fill, pink text.
- Feature card: Deep Graphite, `44px` radius, strong soft shadow.
- Input field: transparent, Slate Gray border, Arial text.

## Reusable Lessons

- Playful can be spacious and sophisticated.
- One vivid interaction color is enough.
- Large rounded dark cards can create contrast without clutter.
- Gradients should act as atmosphere, not every component fill.

## Anti-Patterns

- Do not introduce random chromatic UI colors.
- Do not use arbitrary radius values.
- Do not make body sections dense.
- Do not add shadows outside intended card surfaces.
