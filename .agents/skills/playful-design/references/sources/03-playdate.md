# Playdate — Playful Design Source Notes

Source: https://styles.refero.design/style/2175034b-96d7-417e-886f-ff5a4d8551ae
Original URL: https://play.date
Category scan: Playful Design
Theme: mixed
North star: Lemon Drop Arcade.

## Essence

Playdate is a playful product identity built from bold yellow, blocky layouts,
single-family typography, crisp product imagery, and tactile buttons. It feels
like a handheld console translated into a web system: bright, sturdy, physical,
and instantly recognizable.

## Typography

- Roobert is the sole typeface.
- Sizes from `19px` to `68px`.
- Weights: `400`, `700`, `800`.
- Display: `68px`, line-height `1`.
- Use weight/size, not extra fonts, for hierarchy.

## Layout

- Full-bleed yellow blocks.
- Dramatic central product render in hero.
- Alternates yellow, gray, and charcoal sections.
- Centered content with large vertical sections.
- 3-column game grid.
- Section gap around `68px`.

## Components

- Violet gradient pill CTA, huge `152.19px` radius.
- White button with black text and top border.
- Yellow button with charcoal text and `6px` radius.
- Input field: Paper White, Parchment border, `6px` radius.
- Game grid cards: transparent, no shadow, no radius.

## Reusable Lessons

- A single iconic color can define the entire experience.
- Playful does not need many fonts.
- Physical/tactile button details can make UI feel game-like.
- Product render can anchor loud brand color.

## Anti-Patterns

- Do not introduce extra fonts.
- Do not add complex shadows.
- Do not use random colors outside the defined palette.
- Do not use generic button shapes.
