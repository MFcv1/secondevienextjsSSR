# Duolingo — Playful Design Source Notes

Source: https://styles.refero.design/style/7088d695-362b-4e09-b325-fa8136d4f350
Original URL: https://duolingo.com
Category scan: Playful Design
Theme: light
North star: Playground Starter Kit.

## Essence

Duolingo is gamified learning UI: open white canvas, Duo Green as the core
action/brand color, sky blue secondary links, plump display type, rounded body
type, character illustrations, and tactile 3D-style buttons. It feels energetic
because the content floats in white space and every key action looks pressable.

## Typography

- `feather`: large headlines only.
- `din-round`: body, UI, buttons.
- Body/UI uses distinctive `0.053em` letter spacing.
- Display: `64px`, line-height `1.2`.
- Body: `15px`, line-height `1.4`.

## Layout

- Centered max width around `1140px`.
- Asymmetric hero: illustration left, text/CTA right.
- Generous `80-120px` section spacing.
- Alternating two-column illustration/text sections.
- Vast white canvas; illustrations float.

## Components

- Green headline: feather at `48-64px`, Duo Green.
- Character illustration: organic flat shapes with brand palette.
- Primary CTA: green with solid darker bottom shadow.
- Inline text link: Sky Blue.
- Language flag item: flag + uppercase text.

## Reusable Lessons

- Tactile button shadows make playful UI feel interactive.
- Illustrations should carry most color variety.
- One primary action color keeps gamification clear.
- White space prevents colorful UI from feeling overwhelming.

## Anti-Patterns

- Do not use sharp corners.
- Do not use Feather for small/body text.
- Do not use traditional soft card shadows.
- Do not make text links any color except the defined secondary link.
