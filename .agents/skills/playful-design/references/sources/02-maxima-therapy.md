# Maxima Therapy — Playful Design Source Notes

Source: https://styles.refero.design/style/a122b132-2259-41ca-a301-4468dd17a386
Original URL: https://maximatherapy.com
Category scan: Playful Design
Theme: light
North star: Vibrant, rounded play-world.

## Essence

Maxima Therapy is a saturated play-world. It uses Sunshine Yellow as a dominant
page atmosphere, Action Orange for CTAs, rounded typography, soft cards, and
illustration colors as emotional support. It is maximalist but still systematic:
colors have roles, corners are always rounded, and cards use generous padding.

## Typography

- ABC Diatype Rounded Plus: UI, navigation, buttons, card titles.
- Robuck Rounded: display headlines.
- ABC Diatype Rounded: oversized artistic background/display.
- System sans only fallback/body support.
- Display: `62px`, line-height `0.8`.
- Huge decorative type can reach `214px` or more.

## Layout

- Full-bleed vivid backgrounds.
- Hero is illustration-led and nearly full viewport.
- Centered text blocks over yellow fields.
- Abstract illustrated shapes carry content.
- Section gap around `38px`.
- Card padding around `48px`.

## Components

- Primary Donate Button: Action Orange, white text, `47.62px` radius.
- Ghost arrow button: circular white button.
- Hero card: Action Orange fill, `23.81px` radius, large padding.
- Informational card: Sunshine Yellow fill, rounded.
- Program badges: rounded and compact.

## Reusable Lessons

- Saturated color works when every color has a job.
- Rounded display type can make sensitive domains feel approachable.
- Use illustrations for color variety, not random UI chrome.
- Big padding keeps playful blocks from feeling chaotic.

## Anti-Patterns

- Do not use sharp corners.
- Do not desaturate the palette.
- Do not add new primary colors.
- Do not use shadows for elevation.
