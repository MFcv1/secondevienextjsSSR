# Dia Browser - Soft Gradients Source Notes

## Core read

Dia Browser is the "ethereal gradient glass" archetype: a spacious light
interface where lavender, peach, rose, and amber gradients sit behind frosted
surfaces. The mood is airy and future-facing, but the structure remains simple:
large search/input forms, calm cards, soft borders, pill controls, and enough
ink contrast to keep everything readable.

This source is important because it shows how soft gradients can be immersive
without becoming a generic blob background. The gradients create a field of
light; the product UI still owns the foreground.

## Typography

- Primary: Inter or similar humanist sans.
- Display option: Space Grotesk or a geometric sans fallback.
- Sizes: 14, 15, 16, 18, 20, 24, 30, 36, 48, 64.
- Weights: 300, 400, 500, 600, 700.
- Line heights: display 1.1-1.2, body 1.5-1.7.
- Display tracking: about -0.02em.
- Avoid aggressive condensed type; this style needs air.

## Structure

- Base unit: 4px.
- Density: comfortable to spacious.
- Section gap: around 80px.
- Card padding: 24px.
- Element gap: 16px.
- Button radius: 12px.
- Card radius: 24px.
- Input radius: 16px.
- Default radius: 12px.

## Component lessons

1. Frosted input centerpiece
   - Large rounded input placed in a soft-lit zone.
   - Border should be visible but quiet.
   - Shadow is white/lavender, not black.

2. Translucent pill controls
   - Use subtle white transparency.
   - Pair with a thin border and blur.
   - Active state can tint violet or amber.

3. Gradient text accent
   - Use sparingly for one word or short phrase.
   - Keep body text flat and readable.

4. Soft card glass
   - White cards may be slightly translucent.
   - Use blur only when it improves depth.
   - Do not blur large areas behind important text.

5. Ambient color field
   - Lavender, peach, and rose can mix across a hero section.
   - Keep saturation low enough for premium calm.

## Reusable pattern

Use this source when building:

- browser/search experiences
- AI assistants
- creative productivity tools
- landing pages with a magic-but-calm mood
- onboarding flows that need softness and optimism

## Anti-patterns

- Dark glass panels on a soft light page.
- Too many saturated gradients.
- White text on pastel gradient.
- Blur applied to every card.
- Pill buttons with no active/focus distinction.
- Generic purple-blue gradient without warmth.
