# Sprig - Soft Gradients Source Notes

## Core read

Sprig is the "soft gradient intelligence" archetype: warm cream background,
peach/pink/yellow gradient moments, dark navy text, and saturated blue CTAs.
It is approachable, but it keeps enterprise clarity through strong text
contrast, well-defined cards, and a crisp action system.

This reference is useful when a product wants to say "AI or insights are
helping you" without using cyber neon, black command centers, or childish
pastels. It sits between friendly SaaS and modern intelligence.

## Typography

- Primary: Inter.
- Secondary/technical: JetBrains Mono.
- Headings: dark navy, medium-to-bold, not too decorative.
- Body: 15-17px, spacious line height.
- Labels and data marks can use mono for precision.
- Use readable case and pragmatic product language.

## Structure

- Warm page sections carry the emotional tone.
- White cards keep product modules legible.
- CTAs can use saturated blue for confidence.
- Gradient fields should sit behind "insight" or "AI" moments.
- Use generous spacing around claims, but keep dashboard modules tighter.

## Component lessons

1. Blue action on warm canvas
   - Blue works as a trust signal against cream and peach.
   - It should be the clear CTA, not one of many accents.

2. Insight cards
   - White surface.
   - Navy heading.
   - Slate explanatory text.
   - Small purple/pink/orange accent marks for insight categories.

3. Warm gradient hero
   - Peach/pink/yellow gradient can occupy large hero background.
   - Keep text in navy or black.
   - Place a product preview or dashboard artifact in white above it.

4. Mono precision
   - Use JetBrains Mono for tags, data chips, metrics, or technical labels.
   - Do not use mono for long body copy.

5. Soft segmentation
   - Divide content with white cards and border gray rather than heavy shadows.

## Reusable pattern

Use this source when building:

- customer feedback tools
- AI research products
- analytics or survey tools
- B2B SaaS that needs warmth
- product intelligence dashboards

## Anti-patterns

- Making the cream background too beige.
- Using pastel text colors for body copy.
- Letting blue, purple, pink, orange, and green all compete.
- Replacing product clarity with friendly illustration only.
- Dashboard cards floating with heavy black shadows.
