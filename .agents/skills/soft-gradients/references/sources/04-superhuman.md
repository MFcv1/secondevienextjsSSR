# Superhuman - Soft Gradients Source Notes

## Core read

Superhuman is the "warm dusk productivity" archetype: cinematic sunset
gradients, deep navy shadows, pearl text, and a polished premium rhythm. It is
not a light pastel interface. It uses gradients as dramatic atmosphere around
focused workflow objects.

This source matters because soft gradients do not have to mean white pages.
They can sit inside a dark premium product story when the glow is warm, the
surfaces are restrained, and the typography carries confidence.

## Typography

- Primary: Inter.
- Editorial/display option: Crimson Text.
- Technical option: SF Mono.
- Sizes: 12-72px.
- Weights: 300-800.
- Display line height: around 1.0-1.1.
- Body line height: around 1.5-1.7.
- Display tracking: around -0.03em.

## Component lessons

1. Cinematic gradient hero
   - Dark navy base.
   - Warm coral/amber glow from one side.
   - Foreground object remains sharply readable.

2. Premium productivity card
   - Dark surface or white email panel.
   - Minimal chrome.
   - One strong glow or border highlight.

3. Sunset CTA
   - Coral-to-amber gradient can be the primary CTA.
   - Pair with dark text only when contrast works; otherwise white text.

4. Editorial premium accent
   - A display serif can add human sophistication.
   - Use it for big claims, not for dense UI.

5. Mixed mode hierarchy
   - Dark hero can transition into light content.
   - Reuse warmth across both modes so the page feels continuous.

## Reusable pattern

Use this source when building:

- premium productivity pages
- email and workflow tools
- executive SaaS positioning
- dark hero plus light product sections
- cinematic AI assistant/productivity narratives

## Anti-patterns

- Generic dark blue page with no warm light.
- Red/orange gradients that look like alerts.
- Overusing serif in product UI.
- Low-contrast gray text on navy.
- Large dark cards with no internal hierarchy.
