# Copy - Minimal Design Source Notes

## Core read

Copy is the "strategic blueprint on polished steel" archetype: a muted light
enterprise canvas, precise typography, strict 4px radii, no-shadow hierarchy,
and a single vivid violet for CTAs and active states. It feels minimal because
every visual cue is assigned a job: text, background, border, action, active
state, or process emphasis.

This reference is valuable for minimal B2B/AI SaaS work where the brand still
needs a clear accent and strong conversion path.

## Typography

- Display: ABC Normal or Montserrat fallback.
- Display weights: 500 and 600.
- Display sizes: 24-88px.
- Display letter spacing: -0.02em at 48px+.
- Functional text: Inter.
- Inter weights: 400-700.
- Use ABC Normal for major headings; do not replace with Inter.

## Structure

- Comfortable density.
- Base unit: 4px.
- Section gap: 72px.
- Card padding: 30px.
- Element gap: 16px.
- Button/default radius: 4px.
- Inputs use Cloud Burst and Slate Echo.
- Avoid shadows; depth comes from background and spacing.

## Minimal lessons

1. Strict radius creates identity
   - 4px or 0px only for most UI.
   - Avoid arbitrary rounding.

2. Violet is action, not decoration
   - CTA, active nav, process emphasis.
   - Do not use it for body text.

3. Inputs can blend into canvas
   - Cloud Burst background with Slate Echo outline.
   - Clean path for data entry.

4. Minimal hierarchy is explicit
   - Strong headings, muted body, precise gaps.

5. No-shadow minimal can still convert
   - Use spacing and color roles to direct the eye.

## Anti-patterns

- Shadows for elevation.
- Extra visual accents.
- Violet body text.
- More radii.
- Inter for all major headlines.
- Off-grid spacing.
- Decorative illustrations competing with process clarity.
