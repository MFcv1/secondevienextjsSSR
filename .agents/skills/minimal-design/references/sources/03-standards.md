# Standards - Minimal Design Source Notes

## Core read

Standards is the "high-contrast precision blueprint" archetype: near-white
canvas, black ink, hairline dividers, strict typography, almost no decoration,
and a single orange action color. It is minimal in the hard, exacting sense:
flat, functional, and typographic.

This reference is useful when minimal design must feel decisive instead of
soft. It is for brands that want competence, modern exactitude, and no visual
fluff.

## Typography

- Sole typeface: Soehne-like sans.
- Weights: 400 and 600 only.
- Display around 52px.
- Body around 20px.
- Small captions around 10-14px.
- Letter spacing around -0.01em.
- Use type hierarchy instead of visual decoration.

## Structure

- Full-bleed visual model.
- Large centered headline on Canvas Ice.
- Full-width black block for product/video/showcase.
- Six-column or multi-column grids.
- Cards can have 0px radius and no shadow.
- Ghost cards defined by top/left hairlines.
- Section rhythm around 46px.

## Minimal lessons

1. Flatness can be powerful
   - No shadows.
   - No gradients.
   - No complex backgrounds.

2. One orange accent is enough
   - CTA and critical signal only.
   - Avoid secondary chromatic colors.

3. Borders are structure
   - Use top/left hairlines and dividers.
   - Let geometry organize content.

4. Typography must be exact
   - Same letter spacing everywhere.
   - Few weights.
   - Controlled sizes.

5. Imagery is direct product evidence
   - Screenshots on pure/simple backgrounds.
   - No lifestyle imagery.

## Anti-patterns

- Extra colors.
- Rounded card softness.
- Shadows.
- Decorative gradients.
- Font mixing.
- Inconsistent tracking.
- Wide illustrative scenes.
