# TypeList — Experimental Type Source Notes

Source: https://styles.refero.design/style/76ace78c-94b7-421d-a8fd-47289328458f
Original URL: https://typelist.upstatement.com
Category scan: Experimental Type
Theme: light
North star: Chromatic interactive blocks

## Essence

TypeList makes typography playful through a system of large interactive color
blocks. Unlike display-heavy experimental systems, the type itself is mostly
classic and quiet. The experiment is in the relationship between type, scale,
flat color, and block-based navigation. The result feels like an interactive
typographic exhibit on a white canvas.

## Typography

- `Untitled Sans`: primary UI, body, buttons, headings.
- `Untitled Serif`: occasional phrase or decorative body contrast.
- Sizes: `16px`, `22px`, `24px`.
- Weights: `400`, `500`.
- Line heights: `1.00`, `1.10`, `1.36`.

## Layout

- Full-bleed color sections.
- Header remains sparse and right-aligned.
- Hero headline is centered on a white field.
- Main interaction uses large edge-to-edge color blocks.
- Radius: `0px`.
- Element gap: `30px`.
- Section gap: `48px`.

## Components

- Ghost navigation button: pure text, no background, no border.
- Interactive color block link: full-width block with vivid solid fill, black
  text, square corners, and at least `20px` vertical padding.

## Reusable Lessons

- Color can be structural, not decorative.
- Use flat chromatic blocks as navigation and content.
- Keep type calm when color is loud.
- Reserve vivid colors for large surfaces, not tiny accents.
- Use square geometry to prevent the palette from becoming childish.

## Anti-Patterns

- Do not add shadows.
- Do not turn the vivid colors into small random accents.
- Do not round the blocks.
- Do not add gradients.
- Do not make the layout dense; block-based rhythm needs space.
