# Sociotype — Experimental Type Source Notes

Source: https://styles.refero.design/style/973332dc-4e10-4e90-85d8-3bce9c3cd3ed
Original URL: https://socio-type.com
Category scan: Experimental Type
Theme: light
North star: Editorial White Canvas

## Essence

Sociotype is an expansive editorial typography system. It keeps the interface
almost invisible so type specimens can carry the visual identity. The palette is
achromatic, the radius is zero, the buttons are ghosted or underlined, and the
space is generous. Experimentalism comes from the range and scale of type
families rather than from decorative UI.

## Color Tokens

| Name | Value | Role |
| --- | --- | --- |
| Canvas White | `#ffffff` | Page, card, and footer backgrounds |
| Ink Black | `#000000` | Primary text, borders, active states |
| Medium Gray | `#818181` | Muted text, inactive links, placeholders |
| Light Gray | `#d6d6d6` | Dividers and subtle borders |
| Faded Gray | `#9d9d9d` | Tertiary text and list borders |

## Typography

- `Onsite`: body, navigation, buttons, functional text.
- `Avec Sharp`, `Ceno`, `Meso`, `Gestura`, `Rework`: display specimens.
- Display specimens reach `251px`.
- Functional sizes range from `11px` to `40px`.
- Display letter-spacing: around `0.001em`.
- Functional small text may use `0.015em` to `0.08em` tracking.

## Layout

- Full-bleed structure without fixed max width.
- Large hero type floats over visual or typographic specimens.
- Content often uses two-column editorial composition.
- Very generous vertical rhythm: `120px` section gap.
- Element gap: `12px`.
- Radius: `0px`.
- Card padding: `0px`; content manages itself.

## Components

- Ghost button: underlined text, no fill, no padding.
- Muted ghost button: gray text and gray underline.
- Featured card: transparent, no border, no shadow, no radius.
- Text input: transparent, gray text, thin bottom border.

## Reusable Lessons

- Let the font specimen be the hero.
- Use invisible UI around expressive type.
- Use underlines instead of button boxes.
- Keep all components flat and sharp.
- Give large type enough vertical spacing to breathe.

## Anti-Patterns

- Do not use saturated CTA colors.
- Do not add shadows or elevation.
- Do not round corners.
- Do not cluster content.
- Do not introduce generic system fonts for the typographic voice.
