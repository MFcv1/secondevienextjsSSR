# Egstad — Experimental Type Source Notes

Source: https://styles.refero.design/style/ec17bdec-c8fa-4221-abd6-da717bf38d96
Original URL: https://egstad.com
Category scan: Experimental Type
Theme: light
North star: Type-first Brutalist

## Essence

Egstad is a type-first brutalist system with a warm parchment canvas, dense black
letterforms, enormous display scale, and tactile pill controls. It is visually
minimal but not polite. The typography dominates the viewport, often behaving as
architecture. The UI components feel mechanical, like physical tabs or toggles.

## Color Tokens

| Name | Value | Role |
| --- | --- | --- |
| Canvas Parchment | `#e2e0d9` | Background, borders, dividers, input outlines |
| Inkwell Black | `#252422` | Primary text, dark borders, inverted UI |
| Deepest Black | `#000000` | Secondary text, strong borders, deepest marks |

## Typography

- `Times New Roman`: muted body and secondary information.
- `EG Metaphor`: dominant headlines, architectural display.
- `S85`: specialized navigation labels with wide tracking.
- Display size can reach `399px`.
- Display line-height: `0.97`.
- Display letter-spacing: `-0.055em` at huge scale.
- Small technical labels use `0.100em` tracking.

## Layout

- Contained, centered content with huge typographic hero.
- Hero type can span the full width and overlap portrait/image forms.
- Section rhythm is controlled by big type blocks more than grid complexity.
- Navigation uses a static top bar with pill tabs.
- Element gap: `16px`.
- Section gap: `22px`.
- Card padding: `14px`.

## Components

- Active nav tab: inkwell black fill, parchment text, huge pill radius.
- Inactive nav tab: transparent fill, black text, parchment border.
- Outlined link: S85 at `12px`, `0.1em` tracking, pill radius.
- Large headline: EG Metaphor `399px`, `-0.055em`, 4px underline.
- Body section: Times New Roman `16px`, `1.2` line-height.

## Reusable Lessons

- Use type as physical mass.
- Pair old-system body serif with custom display brutality.
- Use a tiny palette so the letterforms become the event.
- Use extreme pill radius to create tactile contrast against flat type.
- Let underlines and borders be thick enough to feel intentional.

## Anti-Patterns

- Do not add color.
- Do not add gradients or shadows.
- Do not make the display type polite.
- Do not use Times New Roman for display.
- Do not create complex nested layouts.
