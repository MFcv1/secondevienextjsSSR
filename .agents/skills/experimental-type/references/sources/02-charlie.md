# Charlie — Experimental Type Source Notes

Source: https://styles.refero.design/style/34aa811f-6084-484c-b4c0-f587b514e970
Original URL: https://charlielemaignan.com
Category scan: Experimental Type
Theme: dark
North star: High-contrast editorial experimentalism

## Essence

Charlie uses type as an art-gallery object. The system is built from black,
white, intense red, pill navigation, and enormous experimental display type.
The display face carries the entire identity: condensed, theatrical, compressed,
and intentionally hard to ignore. Functional typography stays readable through
NeueHaas, while Brasparz Variable becomes the expressive voice.

## Color Tokens

| Name | Value | Role |
| --- | --- | --- |
| Midnight Ink | `#000000` | Page background, text on white/red surfaces |
| Paper White | `#ffffff` | Text, borders, filled buttons, high contrast |
| Concrete Gray | `#838383` | Muted links, secondary copy, soft borders |
| Alert Red | `#ff0000` | Full-bleed dramatic emphasis blocks |

## Typography

- `NeueHaas`: body, navigation, footer, functional UI.
- `Brasparz Variable`: dominant display typography and stylistic word marks.
- Functional sizes: `19px`, `20px`, `40px`.
- Display sizes: `145px`, `360px`.
- Display line-height: `0.70`.
- Display letter-spacing can reach `-0.079em`.
- Display features: stylistic sets turned off or controlled intentionally.

## Layout

- Full-bleed sections with content constrained near `1306px`.
- Hero can be a full red field with black experimental type.
- Dark sections use white outlined pill controls.
- Section gap: `59px`.
- Element gap: `30px`.
- Buttons use `100px` radius.

## Components

- Ghost nav button: white text, white `2px` border, transparent fill, pill.
- Filled nav button: white fill, black text, pill.
- Footer link: concrete gray text, understated underline on hover.
- Red hero banner: full-bleed red with black display type.

## Reusable Lessons

- Experimental type can be the main image.
- A strict palette makes wild typography feel intentional.
- Use a stable sans-serif to make expressive display type usable.
- Pill controls can soften a brutal black/red editorial system.
- Scale and tracking are the main composition tools.

## Anti-Patterns

- Do not add extra accent colors.
- Do not use soft gradients or textures.
- Do not replace the display face with generic sans.
- Do not make buttons square; the pill shape is part of the contrast.
- Do not reduce the display type too much; this style needs typographic risk.
