# Colab - High Contrast Source Notes

Source: https://styles.refero.design/style/b040360c-38de-4c5f-aada-bd3a67cbedc2  
Site: https://colab.munken.com  
North star: High-contrast theatrical canvas  
Theme: dark

## Extracted Palette

| Name | Hex | Role |
| --- | --- | --- |
| Midnight Canvas | `#000000` | Page backgrounds, heading text, button backgrounds, primary links |
| Paper White | `#ffffff` | Body text, navigation text, button text, ghost button borders |
| Process Red | `#ee1c5b` | Supporting accent for interactive highlights and sharp graphic punctuation |

## Typography

- Primary family: Munken Sans Web.
- Fallback: Inter.
- Weights: 400, 700, 900.
- Sizes observed: 16px, 20px, 40px, 68px.
- Line heights: 1.20, 1.50.
- Letter spacing: 0.0500em.

## Layout Intelligence

- Full-bleed black canvas with centered, high-impact typography.
- Hero sections behave like theatrical posters: dark field, white headline, minimal supporting UI.
- Section rhythm uses generous 80px vertical separation.
- Content stacks are direct, spare, and mostly centered.
- Navigation is minimal and integrated into the canvas.
- Cards are visually naked: no border, no background, no shadow, no radius.

## Components

| Component | Role | Treatment |
| --- | --- | --- |
| Primary Filled Button | Key CTA | Solid black, white text, sharp corners, precise padding |
| Ghost Button | Secondary action | Transparent fill, 1px white border, white text |
| Naked Card | Content wrapper | Transparent, borderless, shadowless, lets content define the edge |
| Site Navigation Link | Top-level navigation | White text on black, minimal state changes |
| Footer Navigation Link | Secondary navigation | White text, generous footer padding |

## Rules Worth Carrying Forward

- Use pure black and pure white without apology.
- Treat the accent red as punctuation, not a palette expansion.
- Keep component corners at 0px.
- Let spacing create drama instead of gradients or shadows.
- Preserve the custom display character of the typeface.
- Never add decorative softness to a stark poster-like system.
