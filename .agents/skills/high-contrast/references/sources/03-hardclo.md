# HARDCLO - High Contrast Source Notes

Source: https://styles.refero.design/style/c3cf4acc-da7a-4496-8c4b-aa89a7c9e36c  
Site: https://hardclo.com  
North star: Monochrome cinematic canvas - stark black and white drama  
Theme: dark

## Extracted Palette

| Name | Hex | Role |
| --- | --- | --- |
| Midnight Noir | `#000000` | Page backgrounds and primary canvas |
| Polar White | `#ffffff` | Interactive text, borders, dividers, large headings |

## Typography

- Display family: Playfair Display.
- Functional family: Haas Grot Disp 65.
- Functional fallback: Helvetica Neue, Arial.
- Functional weights: 400, 500.
- Functional size anchor: 10px.
- Display scale can reach 101px.
- Functional letter spacing: -0.5px.
- Line heights: 1.00, 1.20, 1.40.

## Layout Intelligence

- Split full-bleed composition: black UI canvas paired with large immersive imagery.
- Product/lifestyle imagery is the only major source of color.
- Navigation is compact and list-like.
- Product drama comes from contrast between serif display scale and tiny functional text.
- Components remain sharp, flat, and mostly text-driven.

## Components

| Component | Role | Treatment |
| --- | --- | --- |
| Navigation Link | Primary navigation | 10px Haas Grot, white text, underline hover |
| Underlined Text Link | Secondary interaction | White text with 1px white bottom border |
| Cinematic Image Panel | Product/lifestyle emphasis | Full-height or dominant image against black UI |
| Display Heading | Brand drama | Large serif, white on black, fashion/editorial feeling |

## Rules Worth Carrying Forward

- High contrast can be cinematic rather than technical.
- Pair one large expressive serif with tiny functional sans text for fashion tension.
- Let photography carry color while UI remains achromatic.
- Avoid gray-on-black subtlety; the system needs hard white edges.
- Keep radii at 0px to preserve sharpness.
