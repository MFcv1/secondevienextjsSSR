# Limitless - Editorial Minimal Source Notes

## Core read

Limitless is restrained, achromatic, and product-serious. It shares the
architectural blueprint feel with Legend and Intercom, but it leans more toward
near-gray precision and single-typeface consistency. The violet accent is
present as a brand signal, not as a loud marketing system.

This source is useful for practical editorial minimalism in AI, productivity, or
hardware/software products where the page must feel modern but controlled.

## Typography

- Primary: Greycliff or Inter fallback.
- Weights: 300, 400, 500, 600, 700.
- Sizes: 14, 16, 18, 30, 36, 60.
- Display: 60px, weight 600, line-height about 1.
- Heading: 36px, weight 600.
- Body: 16px, 400, line-height 1.5.
- Body large: 18px, 400.
- Tracking: tight -0.025em across the system.

## Structure

- Base unit: 8px.
- Density: comfortable.
- Section gap: 48-64px.
- Card padding: 16px.
- Element gap: 8px.
- Default radius: 8px.
- Cards: 16px.
- Buttons: 9999px.
- Shadow: one subtle large shadow when needed.

## Component lessons

1. Centered headline block
   - Large, tight, quiet.
   - Uses graphite, not black.

2. Pill ghost button
   - Body Slate text.
   - 9999px radius.
   - Transparent fill.

3. Subtle card container
   - Snowdrift or Porcelain surface.
   - 16px radius.
   - Large soft shadow if elevation is needed.

4. Violet signal
   - Logo, iconography, precise action indicator.
   - Not a broad section background.

5. Single typeface discipline
   - Greycliff everywhere.
   - No decorative font pairing.

## Design lessons

- Achromatic product pages can still feel modern if tracking and type weight are controlled.
- A single violet accent can be a brand signature without becoming a UI theme.
- Pill buttons can coexist with serious typography if the palette stays quiet.
- Slightly gray page backgrounds can feel more architectural than pure white.
- One shadow token is enough.

## Anti-patterns

- Do not introduce bright colors beyond violet.
- Do not deviate from the single typeface system.
- Do not add decorative backgrounds.
- Do not use generic button shapes.
- Do not use glows or heavy elevation.
- Do not loosen the tight tracking that creates the precise tone.
