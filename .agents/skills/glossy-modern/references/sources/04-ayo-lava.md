# Ayo / Lava - Glossy Modern Source Notes

## Core read

Ayo/Lava is the "midnight gradient glow" archetype: a saturated dark interface
with multi-hued gradients, rounded tactile controls, and bright outlines that
feel like light caught on glass. It sits closer to expressive glossy modern
than the strict grayscale command systems.

This source is useful when the brief asks for a futuristic, liquid, energetic
gloss while still needing a readable UI. The hard part is controlling the
gradient so the page does not become cyber neon.

## Typography

- Bold/compact display can work against the dark glow.
- Fine weights add sophistication in secondary copy.
- Use strong text contrast; reserve true white for primary copy.
- Avoid too many font families; the gradient is already expressive.

## Structure

- Dark full-bleed sections.
- Rounded tactile components.
- Sharp-edged panels can coexist with rounded controls.
- Glows should be attached to components or section boundaries.
- High-contrast text should sit away from saturated gradient cores.

## Gloss lessons

1. Multi-hue is allowed, but governed
   - Treat spectrum as atmosphere and edge lighting.
   - Do not turn it into every CTA.

2. Green edge accents feel futuristic
   - Use on tags, focus rings, dividers, or selected outlines.
   - Avoid using neon green for long text.

3. Pink/purple is depth, not decoration
   - Use as shadowed glow or secondary border tone.

4. Dark surfaces preserve readability
   - Keep card interiors black/charcoal.
   - Let gradient live behind or around them.

5. Tactility requires restraint
   - Glow plus border plus gradient plus shadow is too much.
   - Choose two material cues per component.

## Anti-patterns

- Cyber neon overload.
- Spectrum backgrounds behind paragraphs.
- Neon green body copy.
- Random glowing cards with no hierarchy.
- Too many bright accents competing with the product.
