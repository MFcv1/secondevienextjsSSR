# Changelog - Glossy Modern Source Notes

## Core read

Changelog is the "midnight frosted command" archetype: a dark achromatic UI
where depth comes from precise layers, subtle borders, pill controls, and
frosted-glass restraint. It is glossy without being shiny. The polish appears
through calibrated grayscale, crisp type, compact spacing, and button surfaces
that feel lightly sealed behind glass.

This reference is important because it shows the quiet side of glossy modern:
no colorful glare, no 3D plastic, no giant hero gradients. It relies on
material subtlety.

## Typography

- Primary: Inter Variable.
- Technical: Berkeley Mono.
- Display: 48px, weight around 510, line-height 1.
- Heading large: 32px, weight around 590.
- Body: 16-17px, weight 400.
- Letter spacing: about -0.01em across primary UI.
- Code/technical text uses mono with slightly tight tracking.

## Structure

- Base unit: 4px.
- Density: compact.
- Section gap: 24px.
- Card padding: 16px.
- Element gap: 8px.
- Cards: 8px radius.
- Search: 100px radius.
- Buttons: 9999px radius.
- Interactive micro radius: around 4px.

## Gloss lessons

1. Use grayscale as material
   - Each layer is a slightly different black/charcoal.
   - Avoid loud color; contrast creates polish.

2. Borders replace shadows
   - 1px graphite borders define glass edges.
   - Heavy drop shadows would break the command-center precision.

3. Pill controls carry tactility
   - Buttons and filters are fully rounded.
   - Cards remain modestly rounded.

4. Compact rhythm feels engineered
   - Glossy modern can be dense.
   - Use tight rows and precise type to make surfaces feel intentional.

5. Text contrast is the luxury
   - Primary text should be nearly white.
   - Muted text should still be readable.

## Anti-patterns

- Neon accents everywhere.
- Purple/blue gradient pasted behind every block.
- Heavy glass blur on content rows.
- Rounded cards with huge empty spacing.
- Dark gray text that disappears.
- Shiny reflections with no functional hierarchy.
