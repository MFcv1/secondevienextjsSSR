# Dia Browser - Glossy Modern Source Notes

## Core read

Dia Browser is the "prism on white stationery" archetype: an almost
achromatic light canvas where color appears as refracted spectrum, translucent
surfaces, and frosted-glass depth. The page feels glossy because light passes
through the interface rather than sitting on top of it.

This source is the light counterpart to the dark obsidian systems. It proves
that glossy modern can be delicate, neutral, and airy.

## Typography

- Primary/display: ABC Oracle or a light high-contrast display substitute.
- Display: 72px and 54px at weight 300.
- Heading: 50px, weight 400.
- Body: 14-18px, weight 400.
- Labels/buttons: 14-16px, weight 500.
- Display tracking: -0.04em.
- Heading tracking: -0.02em.

## Structure

- Base unit: 8px.
- Density: spacious.
- Max width: 1200px.
- Section gap: 80-120px.
- Card padding: 32px.
- Element gap: 15-20px.
- Cards/buttons: 30px radius.
- Containers: 40px radius.
- Avoid radius below 10px.

## Gloss lessons

1. Spectrum is brand, not UI fill
   - Gradient appears as light/refraction.
   - Do not use saturated stops as button backgrounds.

2. Neutral buttons preserve elegance
   - Pebble gray buttons keep the hierarchy quiet.
   - Chromatic CTAs would break the stationery feeling.

3. Frosted surfaces are bright and legible
   - White at high opacity plus blur.
   - Minimal shadow, no heavy lift.

4. Thin display type can feel like etched glass
   - Weight 300 large headings make the UI airy.
   - Tight tracking keeps them from falling apart.

5. Large radii create softness
   - Cards and buttons are rounded, but not toy-like.

## Anti-patterns

- Saturated gradient button fills.
- Spectrum gradient as text.
- Sharp corners.
- Heavy shadows.
- Pure white sterile canvas with no prism warmth.
- Dark glass components on the light stationery page.
