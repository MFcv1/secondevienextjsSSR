# Raycast - Glossy Modern Source Notes

## Core read

Raycast is the "obsidian backlit instrument" archetype: a near-black canvas
where UI surfaces emerge like backlit glass panels. The gloss is tactile rather
than decorative. Keyboard keys, pills, and controls feel pressable because of
layered inner highlights, dark shadows, and precise rings.

This source teaches that glossy modern can be physical without becoming
skeuomorphic. The page feels like a precision launcher: dark, dense, luminous,
and ready for keyboard-driven action.

## Typography

- Primary: Inter.
- Code: Geist Mono.
- Display: 64px/56px range, weights 400-600.
- Body: 14-18px.
- Display tracking is tight/negative.
- Micro labels and badges use positive tracking.
- Mono labels use positive tracking for command readability.

## Structure

- Base unit: 8px.
- Max width: 1200px.
- Section gap: 80px.
- Card padding: 24px.
- Element gap: about 15px.
- Buttons: 8px radius or pill.
- Cards: 11px radius.
- Modals: 16px radius.
- Large cards: 20px radius.

## Gloss lessons

1. Pressable surfaces
   - Use inner white highlights and dark lower shadows.
   - Buttons should feel like hardware keys.

2. Near-white CTA on black
   - Primary action can be neutral, not chromatic.
   - This feels mature and instrument-like.

3. Atmospheric glows stay behind content
   - Blue/violet radial glows are section atmosphere.
   - Surfaces remain neutral charcoal.

4. Two-register typography
   - Tight display type creates pressure.
   - Spaced micro labels create scannability.

5. Color is status, not decoration
   - Red is brand/status.
   - Mint is state.
   - Gradients are atmospheric.

## Anti-patterns

- Colored surface backgrounds.
- Over-rounding cards above 20px.
- Saturated chromatic CTA when neutral action is stronger.
- Cyberpunk blue/purple glow everywhere.
- Flat black with no tactile control shadows.
