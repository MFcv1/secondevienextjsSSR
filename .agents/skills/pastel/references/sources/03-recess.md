## Typography
### Sharp Grotesk Web (ALL three weights)
- Sub: Montserrat. Weights: 400, 500, **700**
- Sizes: 14, 15, 16, 18, 20, 24, 60px
- **One typeface, three roles** (body 400, subhead 500, headline 700)

## Spacing & Shapes
- Base 4px, **density: spacious**
- Section gap: **48-80px**
- Element gap: 8px
- Scale: 4, 8, 12, 16, 20, 24, 28, 32, 36, 40, 48, 56, 84, 144

| Element | Value |
|---------|-------|

## Components

### Primary Accent Button
**Lavender Mist bg, Sky Violet text, 1px Sky Violet border, 0px radius, 24px/36px padding**.

### Outlined Text Button
Transparent bg, Sky Violet text + 1px border, 0px radius, 12px padding.

### Ghost Icon Button
Transparent, Sky Violet color, no border, **50% radius (circle)**, 0px padding.

### Text Input Light
Transparent bg, Sky Violet text, **1px Sky Violet bottom-border only**, 0px radius, 12px vertical padding.

### Text Input Dark
Transparent, white text, 1px white bottom-border.

## Imagery
**Vibrant product photography** of Recess cans/powders with fresh fruits & nature. Products artfully arranged on gradient backgrounds. **Abstract soft 3D cloud graphics float around UI.** Minimal Sky Violet/Jet Black icons. Aspirational lifestyle visuals.

## Layout
Full-bleed structure. Hero: large gradient/photo bg + bold centered headline + prominent CTA. Sections alternate full-bleed gradients ↔ contained rows. Centered stack → two-column image-left/text-right. Generous vertical spacing. Sticky top bar with shop/cart/user icons.

## Motion Notes
- **3D clouds float around UI elements** (signature whimsical motion)
- Pastel gradient backgrounds may slow-drift
- Soft fade transitions
- Button hover with Lavender Mist intensity shift
- Calm, dreamlike — not aggressive

## Do's
- Sharp Grotesk Web only (400 body, 500 subhead, 700 headline)
- Sky Violet (#25385b) for ALL primary text & button text
- Lavender Mist (#a2b0ff) for primary button bg + Sky Violet text
- **0px radius on buttons & cards** (crisp contrast to organic forms)
- Sky Glow A gradient for hero/key blocks
- 24px/36px padding on prominent CTAs
- 8px element gap between interactive

## Don'ts
- No Adobe Blank for display text (placeholder only)
- No strong dark shadows on cards
- No saturated colors for large blocks (accents only)
- No deviation from 0px radius (except 50% circle icons)
- No tight section spacing (48-80px sectionGap minimum)
- No dark bg for body text (Sky Violet on light only)
- No additional typefaces

## Similar Brands
Olipop, Kin Euphorics, Dirty Lemon, Moon Juice, Starface

## Key Insight
**Pastel + sharp corners is the trick.** Most expect pastel = rounded, but Recess uses 0px radius for "intentional contrast with organic product forms". Lavender Mist button + Sky Violet text is the signature combo. Three sky-glow gradients create the "cloud dreamscape" — the hero gradient (cyan→violet→rose) is the visual identity.
