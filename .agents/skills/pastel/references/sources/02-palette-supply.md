## Typography (Custom Fonts)
### PPSupply (Primary UI)
- Sub: Segoe UI, Roboto. Weights: **100, 300, 400** (no bold)
- Sizes: 13, 15, 16, 18px. Letter-spacing: -0.04em (small), 0.02em (medium)

### esbuild (Display)
- Weight: 400. Size: **64px**. Letter-spacing: -0.03em

### PPSupplyMonoLight (Captions)
- Weight 300. Size 12px

## Imagery
**Abstract geometric color blocks with rounded corners.** No photos, no complex illustrations. Minimalist outlined icons. Decorative atmospheric brand accents, moderate density.

## Layout
Full-bleed (no max-width). Hero: centered text typography over abstract color blocks. 72px section gaps. Alternating visual blocks or centered stacks. Multi-column card grids for features. Comfortable spacious density.

## Motion Notes
- Subtle hover state transitions
- Color blocks may animate/rotate as decorative
- Gentle fade reveals on scroll
- No aggressive motion — soft and friendly tone

## Do's
- Canvas (#f2f0e9) default warm off-white bg
- Paper White cards on top
- 100px radius on ALL buttons/inputs (signature)
- Indigo Punch ONLY for primary CTAs
- PPSupply for all UI text (custom!)
- esbuild for 64px display
- 72px section spacing, 8px element gap

## Don'ts
- No multiple vivid accents juxtaposed (curate palette blocks)
- No sharp corners or radii < 5px
- No system defaults instead of PPSupply
- No strong drop shadows
- No arbitrary button colors
- No default link blue

## Similar Brands
Canva, Mailchimp, Intercom, Stripe Atlas

## Key Insight
**Warm-canvas + multi-earthy-palette + 100px-pills.** The signature is the 100px radius EVERYWHERE on buttons/inputs (full pills). Warm off-white `#f2f0e9` canvas instead of pure white. Multiple decorative earth-tone accents (terracotta, harvest gold, sage, desert rose) for content cards, but ONE blue (Indigo Punch) for CTAs. Light fonts (100, 300) create airy quality.
