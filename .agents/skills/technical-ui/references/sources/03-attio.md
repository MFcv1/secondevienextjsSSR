# Attio — Technical UI Source Notes

Source: https://styles.refero.design/style/08c8700c-f278-42bc-812e-f60dc6ce996e
Original URL: https://attio.com
Category scan: Technical UI
Theme: light
North star: Precision Digital Toolkit.

## Essence

Attio is a high-end technical toolkit UI with strong monochrome discipline,
soft editorial serif headlines, neutral Inter UI, and precise borders. It feels
like a carefully designed instrument: minimal but detailed, sharp but not cold.
Color is almost absent, with interaction handled through black, white, borders,
and occasional blue focus/link states.

## Typography

- `Tiempos Text`: large headlines only, `28px+`.
- `Inter Display`: large technical sans headings.
- `Inter`: UI, body, navigation, buttons.
- Uses `ss03` stylistic set.
- Display reaches `64px`, line-height `1.07`.

## Layout

- Centered max-width model around `1440px`.
- Minimal centered hero headline stack.
- Stacked sections and simple two-column layouts.
- Product UI demonstration is the main visual proof.
- Sticky top bar.
- Section gap around `96px`.

## Components

- Primary CTA: Ink background, white text, `10px` radius.
- Secondary CTA: white fill, slate border, `10px` radius.
- Header nav button: transparent, metal text, ash hover.
- Feature tabs: active bottom border, `0px` radius.
- UI frame card: white, stone border, `8px` radius, subtle shadow.

## Reusable Lessons

- A serif display face can humanize clinical technical UI.
- Borders can replace colorful decoration.
- Product UI frames should be the visual centerpiece.
- Use color sparingly for focus and links only.

## Anti-Patterns

- Do not use serif body copy.
- Do not color headlines or primary buttons.
- Do not shadow simple controls.
- Do not vary radii casually.
