# ToDesktop — Technical UI Source Notes

Source: https://styles.refero.design/style/dd89ce6c-f0aa-4ca8-bd63-19dcd81920a7
Original URL: https://www.todesktop.com
Category scan: Technical UI
Theme: mixed
North star: Digital engineering lab.

## Essence

ToDesktop is a mixed light/dark engineering lab aesthetic. It uses deep blue
hero surfaces, crisp white frosted cards, electric blue CTA accents, Aeonik
headings, Inter UI, and Geist Mono for code/terminal moments. The system balances
serious technical depth with clear conversion and product marketing.

## Typography

- Inter: UI, body, navigation, buttons, tables.
- Aeonik Pro: major headings.
- Geist Mono: code, terminal, technical labels.
- Display reaches `74px`, line-height `1.08`.
- Mono uses slight positive letter-spacing.

## Layout

- Mixed light/dark sections.
- Deep dark hero or engineering-lab surface.
- White frosted feature cards.
- Code/terminal proof modules.
- Section gap around `64px`.
- Element gap around `8px`.

## Components

- Primary action: Electric Blue, white text, full pill radius.
- Ghost button: transparent, black text, silver border, `0px` radius.
- Navigation link button: subtle border, `6px` radius.
- Dark hero card: Midnight Void, `24px` radius, inner/outer shadows.
- White frosted card: white with opacity, `20px` radius, soft shadows.
- Info badge: white fill, Sky Blue text, pill radius.

## Reusable Lessons

- Technical UI can combine dark engineering depth with light content cards.
- Electric Blue should be used only for primary action and essential highlights.
- Code/terminal content needs a dedicated mono font.
- Mixed-mode pages need strict contrast and section logic.

## Anti-Patterns

- Do not introduce new saturated colors.
- Do not use harsh black shadows.
- Do not mix radius systems casually.
- Do not use generic fonts in code/heading roles.
