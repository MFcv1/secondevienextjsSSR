# Empower - Vibrant Accents Source Notes

## Core read

Empower is the dark-finance bright-button archetype. It uses a deep dark canvas,
strong condensed typography, and one optimistic yellow action color. The brand
switches between dark command sections and clean light sections, but the yellow
remains the active spark across both modes.

This source is useful when vibrant accents should be high-contrast, financial,
and direct rather than playful or gradient-heavy.

## Typography

- Body/nav: GTAmericaExtended-like.
- Body weights: mostly 400.
- Body sizes: 11, 13, 16, 18, 20, 21, 26.
- Body tracking: tight negative values.
- Display: Gravity weight 900 or EmpowerSerif weight 600.
- Display sizes can reach 96px and beyond.
- Headline voice: condensed, direct, loud, financial.

## Structure

- Theme: mixed dark and light.
- Base unit: 8px.
- Density: comfortable.
- Section gap: 32px.
- Card padding: 24px.
- Element gap: 8px.
- Cards: 24px radius.
- Modules: 16px radius.
- Buttons: effectively full pill.
- No heavy shadows; shape and color do the work.

## Component lessons

1. Yellow primary pill
   - Fill #e4e24e.
   - Night Sky text.
   - Large pill radius.
   - Works on dark and light sections.

2. Dark ghost button
   - Transparent fill.
   - Cloud Whisper text/border.
   - Pill radius.

3. Light ghost button
   - Transparent fill.
   - Night Sky text/border.
   - More padding, still pill.

4. Dark photo card
   - Charcoal Card fill.
   - 24px radius.
   - No shadow.
   - Lifestyle/financial imagery can humanize.

5. Muted yellow content card
   - Soft highlight without full action intensity.
   - Use for callouts, not every card.

## Design lessons

- One bright accent can carry an entire mixed light/dark brand.
- Yellow should be a background/action color, not body text.
- Strong custom type keeps a simple color system from feeling generic.
- Alternating dark/light sections can be clean if the accent stays constant.
- Cards can be friendly through radius, not shadow.

## Anti-patterns

- Do not introduce other saturated accents.
- Do not use Button Yellow for text.
- Do not use generic system fonts for headlines.
- Do not add heavy drop shadows.
- Do not break pill button logic.
- Do not overuse thin light lines on dark sections.
